Storage Configuration

Choose where to store your backups. Local storage saves files on the server, while remote options (FTP, SFTP, S3) store them externally.

Backup Options

Configure what to include in your backups:

  • Files: Uploaded documents and media
  • Database: All application data
  • Compression: Reduces backup size
  • Encryption: Secures sensitive data
Schedule

Set up automatic backups:

  • Daily: Runs at specified time
  • Weekly: Runs on chosen day
  • Monthly: Runs on chosen date
Retention

Manage backup storage:

  • Retention days: How long to keep backups
  • Max backups: Maximum number to store
